Typical workflow

  1. Type a rough app idea.
  2. Generate the PowerShell command.
  3. Paste it into PowerShell to append a `[READY]` queue item.
  4. Open `queue.md` and sanity-check the generated queue item.
  5. Run the standard or production pipeline.
  6. Inspect the app's `AGENT_REPORT.md` when finished.
